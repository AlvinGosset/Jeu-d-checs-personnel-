Le jeu ce lance et ce joue dans le terminal de votre ordinateur.
Afin de jouer, lancez tout d'abord votre terminal dans le dossier ou vous avez mis les fichiers que vous avez téléchargé depuis ce dépôt github, exécutez la commande suivante : javac *.java
puis exécutez cette commande : java Main
la commande java Main vous permet de lancer le jeu.
